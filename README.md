## Markdown

Markdown es un lenguaje de marcas pensado para la documentacion y para ser convertido a otros formatos.

### Cabeceras

Las cabeceras se marcan con tantas almoadillas como nivel de cabecera: # es h1; ## es h2;.... asi hasta h6
